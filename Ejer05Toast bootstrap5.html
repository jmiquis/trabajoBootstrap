Ejer05: Toast

Toast es mensaje de alerta ligero y fácilmente personalizable, con el cual podemos, opcionalmente, envíar notificaciones a los usuarios/visitantes de una web.
Este se muestra por un par de segundos al producirse un evento(ej: clicar botones, enviar formularios...ect).

Para crear un Toast necesitamos usar unca clase: .toastclase y dentro de ella podemos dividir la estructura en la cabecera usando .toast-headery y en cuerpo usando .toast-bodydentro.
Por defecto los Toast están ocultos de forma predeterminada, por lo que para mostrarlos necesitamos usar la clase .showclase. Por otra parte para cierrarlo, requerimos de un booton al que debebemos agreguar: data-bs-dismiss="toast".

Toast admite multiples personalizaciones/configuraciones:

  • HUBICACION: puede situarse donde uno quiera, incluso admite configuración con flexbox.
  • COLOR: esquemas divididos en base a color y fondo.
  • ACCESINILIDAD: deben estár envueltos en una aria-liveregión xxxx
